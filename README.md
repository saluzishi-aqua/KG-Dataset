# Data Availability for  

A Multimodal Large Language Model and Knowledge Graph–Driven Approach for Intelligent Product Service

## Overview

This document provides access information for the datasets used in the study entitled  
**“A Multimodal Large Language Model and Knowledge Graph–Driven Approach for Intelligent Product Service.”**

The shared data support the reproducibility of experiments reported in the paper, including multimodal perception, knowledge graph construction, and end-to-end Fault–Action decision-making evaluation.

## Contents

The shared dataset includes the following components:

- **Visual data**
  - Object detection and image classification datasets for railway equipment inspection
- **Acoustic data**
  - Audio samples for equipment operating state recognition
- **Time-series data**
  - Pantograph current signals for operating state classification
- **Textual data**
  - Processed technical documents and extracted entities and relations for knowledge graph construction
- **Evaluation data**
  - Samples used in the end-to-end Fault–Action decision-making task

Detailed descriptions of data preprocessing, experimental settings, and usage are provided in the paper.

---

## Usage Notes

- The dataset is intended **for research and academic use only**.
- Some industrial data have been anonymized or processed to remove sensitive information.
- Users are requested to follow the usage terms of the hosting platforms.
